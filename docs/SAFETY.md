# AblePath Safety Baseline

High-risk actions must never execute directly from model text.

## High-Risk Actions

The following require explicit confirmation before execution:

- sending messages
- deleting content
- purchasing or paying
- publishing publicly
- submitting forms
- granting remote assistance

## Privacy Defaults

- Store activity, screenshots, and configuration locally.
- Keep screen viewing permissions off by default for caregivers.
- Do not store third-party AI account passwords.
- Emergency events are retained for audit.
